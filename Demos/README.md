# Demos

Die Demos sind nach technischen Themenblöcken gegliedert und folgen [`.ai/DEMO_CONTRACT.md`](../.ai/DEMO_CONTRACT.md).

| Ordner | Themen |
|---|---|
| `00_Framework/` | Preflight, Datengeneratoren, Messung und Cleanup |
| `01_Storage_Pages_Log/` | Pages, Allocation, Buffer Pool und Transaction Log |
| `02_Optimizer_Statistics_Plans/` | Optimizer, Statistiken, Plans und Memory Grants |
| `03_Query_Patterns/` | T-SQL-Muster und typische Performancefallen |
| `04_Rowstore_Columnstore/` | Indexdesign, Maintenance und Columnstore |
| `05_Concurrency_Isolation_TempDB/` | Locks, Blocking, Isolation, Versioning und TempDB |
| `06_CPU_Memory_IO_Waits/` | Ressourcen, Scheduler und Wait-Methodik |
| `07_Query_Store_Extended_Events/` | Historische Diagnose und Ereigniserfassung |
| `08_Infrastructure/` | Begleitskripte für externe Laborszenarien |

Neue Demos werden erst nach Vergabe einer stabilen Demo-ID angelegt.
