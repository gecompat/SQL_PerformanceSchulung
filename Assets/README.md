# Assets

Hier liegen wiederverwendbare, firmenneutrale Abbildungen, Diagrammquellen und weitere Präsentationsressourcen.

Logos, Markenauftritte, reale Screenshots, personenbezogene Daten und unklare Drittanbieter-Assets sind nicht zulässig. Für jedes externe Asset müssen Herkunft, Nutzungsrecht und erforderliche Attribution dokumentiert sein.
