# Research

Hier werden fachliche Prüfungen, Primärquellen und versionsbezogene Abgrenzungen dokumentiert.

Jeder Eintrag trennt:

- dokumentiertes Produktverhalten,
- empirisch gemessene Beobachtung,
- daraus abgeleitete Vermutung oder Empfehlung.

Quellen erhalten Titel, Herausgeber, Veröffentlichungs- oder Aktualisierungsdatum, URL und Abrufdatum. Widersprüchliche oder veraltete Quellen werden sichtbar eingeordnet.
